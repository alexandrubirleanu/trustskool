# Duncan Rogoff and Claude Code Club: Inside Skool's 7,300-Member AI Coding Community

Claude Code Club is a 7,360-member Skool community teaching people how to use Anthropic's Claude Code to build real tools, websites, apps, and content from day one, rather than treating AI coding as a theoretical skill. It's priced at $9/month for current members, with the community stating the price rises to $69/month once it reaches 7,400 members [Skool](https://www.skool.com/claudecodeclub/about).

## Who runs it

The community is founded and run by Duncan Rogoff, who also teaches Claude Code on his own YouTube channel [YouTube](https://www.youtube.com/@duncanrogoff) and has a LinkedIn presence under "The Build Room" [LinkedIn](https://www.linkedin.com/in/duncanrogoff/), where he posts about Claude Code setups and AI-native development workflows. According to the community's own about page, Rogoff has experience at Apple, PlayStation, and Nissan, and the page describes him as a Forbes 30 Under 30 recipient and Y Combinator alumnus whose community grew to a 200,000+ audience within 12 months. It's worth flagging that these specific credentials (the Forbes and YC claims) appear in the community's own marketing copy; this research did not find independent, third-party confirmation of Rogoff on an official Forbes 30 Under 30 list, so readers should treat those specific claims as self-reported rather than independently verified, while his YouTube and LinkedIn presence as a real, active Claude Code educator is directly verifiable.

## What members actually get

Claude Code Club is built around practical output: members learn to build websites, apps, content systems, and internal tools using Claude, with templates and step-by-step guidance rather than abstract AI theory. The community also says its instructor roster includes operators who've built AI tools for large clients and previously worked at companies like Apple, PlayStation, and Charles Schwab — positioning the group as run by practitioners currently building with the tool, not just people who tried it once and made a course.

## Why it's grown

Claude Code itself exploded in popularity through 2026 as developers and non-developers alike started using it to ship real software, and Skool has seen a corresponding wave of "Claude Code" branded communities launch in a short window — Claude Code Academy, Claude for Founders, Claude Code Architects, and several others are all competing for the same audience searching for structured guidance on the tool. Claude Code Club differentiated itself early with an aggressive, member-count-triggered pricing model (locking in low pricing for early joiners before a steep jump), which creates urgency and has likely accelerated its growth relative to flatter-priced competitors. Rogoff's own visibly active YouTube presence — where he's demonstrably still building and experimenting with Claude Code rather than only teaching from a script — is the kind of "still in the arena" credibility that tends to matter more in fast-moving AI tooling niches than formal credentials.

Given how crowded the Claude Code education space has become on Skool in a short period, prospective members should compare a few of these communities directly on current activity and member reviews rather than picking based on marketing copy alone.
